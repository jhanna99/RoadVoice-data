# OpenStreetMap Data Attribution

## Data Sources

RoadVoice uses location data from multiple sources to provide information about businesses and points of interest:

### OpenStreetMap (OSM)

Some location data in RoadVoice is derived from [OpenStreetMap](https://www.openstreetmap.org/), a collaborative project to create a free editable map of the world.

**Attribution:** Data from OpenStreetMap is made available under the [Open Database License (ODbL)](https://opendatacommons.org/licenses/odbl/1.0/).

- OpenStreetMap data is copyright OpenStreetMap contributors
- Map data available at [openstreetmap.org](https://www.openstreetmap.org/)
- License: [ODbL 1.0](https://opendatacommons.org/licenses/odbl/1.0/)

The following brands in RoadVoice use OpenStreetMap data:
- 7 Brew
- Bass Pro Shops
- Biggby Coffee
- Cabela's
- Duluth Trading
- Jeremiah's Italian Ice
- Mochinut
- Roller Skating Rinks

### AllThePlaces

Additional location data is sourced from [AllThePlaces](https://www.alltheplaces.xyz/), a project that extracts location data from official store locator websites.

- AllThePlaces data is released under the [CC-0 (Public Domain)](https://creativecommons.org/publicdomain/zero/1.0/) license

### Other Sources

Some brand data is aggregated from publicly available official sources.

## Contributing

If you notice incorrect or missing location data:
- For OpenStreetMap data, you can contribute corrections directly at [openstreetmap.org](https://www.openstreetmap.org/)
- For other data issues, please contact us

## License Compliance

RoadVoice complies with the ODbL license by:
1. Providing this attribution notice
2. Making it clear which data comes from OpenStreetMap
3. Not claiming ownership of the OSM-derived data

For questions about data licensing, please refer to the [ODbL FAQ](https://wiki.openstreetmap.org/wiki/Open_Database_License/FAQ).
